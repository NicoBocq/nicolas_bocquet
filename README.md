# Nicolas Bocquet - Portfolio

Minimalist, typography-focused portfolio.
**Live Demo:** [https://nicobocq.github.io/nicolas_bocquet](https://nicobocq.github.io/nicolas_bocquet)

## Tech Stack

-   **HTML5 / CSS3** (Flexbox, Grid, Variables)
-   **Vanilla JS** (Dynamic content, Dark mode)

## Structure

-   `js/data.js`: Centralized content (Experience, Skills, Profile).
-   `resume.html`: Print-optimized CV.

## Deployment

Push to GitHub and enable **GitHub Pages** (Source: `main`).
